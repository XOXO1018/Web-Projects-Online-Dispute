## 📁 生成文件（与论文格式一致）

### 1. chokepoint_systemic_trade_risk.csv (757 KB)
**内容**: 系统性贸易风险数据
- **记录数**: 852 条（国家 × 咽喉要道）
- **关键字段**:
  - `iso3`, `canal`: 国家代码和咽喉要道
  - `v_canal`, `q_canal`: 通过咽喉要道的贸易量/值
  - `v_canal_import/export`: 进口/出口贸易
  - `likelihood_*`, `timescale_*`, `severity_*`: 8 种风险参数
  - `trade_at_risk_*_v/q`: 各风险类型的贸易风险
  - `revenue_at_risk_*`: 收入风险
  - `trade_impacted_*`: 受影响贸易

### 2. chokepoint_systemic_economic_risk.csv (814 KB)
**内容**: 系统性经济风险数据
- **记录数**: 852 条
- **关键字段**:
  - 每种风险类型的成本分解:
    - `delay_USD_*`: 延误成本
    - `reroute_USD_*`: 改道成本
    - `lost_USD_*`: 损失成本
    - `revenue_USD_*`: 收入损失
    - `war_premium_USD_*`: 战争溢价 ⭐ 新增
    - `shipping_price_USD_*`: 航运价格影响 ⭐ 新增
    - `total_loss_USD_*`: 总成本
  - 汇总成本:
    - `total_loss_USD`: 总经济成本
    - `total_war_premium_USD`: 总战争溢价
    - `total_shipping_price_USD`: 总航运价格影响

### 3. country_import_dependency.csv (5.4 KB)
**内容**: 国家进口依赖度
- **记录数**: 71 个国家
- **关键字段**:
  - `iso3`, `country_name`, `region`
  - `v_canal_import`: 通过咽喉要道的进口贸易
  - `v_total_import`: 总进口贸易
  - `import_dependency`: 进口依赖度 (%)

### 4. country_export_dependency.csv (5.4 KB)
**内容**: 国家出口依赖度
- **记录数**: 71 个国家
- **关键字段**:

  - `v_total_export`: 总出口贸易
  - `export_dependency`: 出口依赖度 (%)

### 5. risk_analysis_dashboard.png (308 KB)

### 关键算法

#### 1. 贸易风险计算
```python
trade_at_risk = trade_volume × (1/likelihood) × (timescale/365) × severity
```

#### 2. 战争溢价计算 (新增)
```python
war_premium = revenue × premium_rate × (1/likelihood) × severity
# premium_rate: geopolitical=5%, conflict=8%, terrorist=3%, etc.
```

#### 3. 航运价格影响 (新增)
```python
shipping_price_impact = reroute_cost × market_multiplier
# market_multiplier = 1.5 (市场紧张时的放大效应)
```

#### 4. 贸易依赖度
```python
import_dependency = v_canal_import / v_total_import
export_dependency = v_canal_export / v_total_export
```

#### 5. 市场集中度 (HHI)
```python
HHI = Σ(market_share²)
# market_share = chokepoint_trade / total_trade
```

## ⚠️ 数据说明

### 当前使用模拟数据
- **国家数据**: 71 个主要贸易国家（基于真实 GDP 和人口）
- **贸易流量**: 基于 GDP 的估算模型（非真实贸易数据）
- **风险参数**: 基于论文和历史统计的模拟值

### 依赖度>100% 的原因
部分国家的进口/出口依赖度超过 100%，这是因为：
1. 使用 GDP 比例估算贸易流量，而非真实海关数据
2. 简化模型未考虑双边贸易协定和地理约束
3. 某些小国的"总贸易"估算偏低

### 升级到真实数据的建议
如需与论文完全一致的结果，建议：
1. 从 **UN Comtrade** 下载真实贸易数据
2. 从 **UNCTAD** 下载海运贸易数据
3. 替换 `create_country_chokepoint_trade()` 函数的数据源
