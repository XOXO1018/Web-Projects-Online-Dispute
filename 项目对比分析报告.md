**关键列** (每个风险类型都有以下列):
- `delay_USD_*`: 延误成本
- `lost_USD_*`: 损失成本
- `revenue_USD_*`: 收入损失
- `reroute_USD_*`: 改道成本
- `war_premium_USD_*`: 战争溢价 ⚠️ **缺失**
- `shipping_price_USD_*`: 航运价格影响 ⚠️ **缺失**

**功能**: 各国对咽喉要道的贸易依赖度
- 出口依赖度
- 进口依赖度

#### 1.1 添加战争溢价和航运价格影响

```python
def estimate_war_premium(df, risk):
    """
    计算战争溢价
    基于风险严重程度和地缘政治紧张程度
    """
    # 战争溢价 = 贸易价值 × 风险溢价率
    premium_rate = {
        'geopolitical': 0.05,   # 5% 溢价
        'conflict': 0.08,       # 8% 溢价
        'terrorist': 0.03,      # 3% 溢价
        'piracy': 0.02,         # 2% 溢价
    }
    
    base_rate = premium_rate.get(risk, 0.01)
    df[f'war_premium_USD_{risk}'] = (
        df['revenue_USD'] * base_rate * 
        (1 / df[f'likelihood_{risk}'].replace(0, np.nan)) *
        df[f'severity_{risk}']
    )
    return df

def estimate_shipping_price_impact(df, risk):
    """
    计算航运价格影响
    基于改道距离和市场紧张程度
    """
    # 航运价格影响 = 改道成本 × 市场乘数
    market_multiplier = 1.5  # 市场紧张时的放大效应
    
    df[f'shipping_price_USD_{risk}'] = (
        df[f'reroute_USD_{risk}'] * market_multiplier
    )
    return df
```

#### 2.1 贸易依赖度分析

```python
def calculate_country_dependency(country_df, chokepoint_df):
    """
    计算各国对咽喉要道的贸易依赖度
    
    依赖度 = 通过咽喉要道的贸易 / 总贸易
    """
    dependency = country_df.merge(chokepoint_df, on='iso3')
    dependency['import_dependency'] = (
        dependency['v_canal_import'] / dependency['v_total_import']
    )
    dependency['export_dependency'] = (
        dependency['v_canal_export'] / dependency['v_total_export']
    )
    
    return dependency
```

#### 2.2 系统性风险评估

```python
def calculate_systemic_risk(df):
    """
    计算系统性风险指标
    
    包括:
    - 网络效应风险 (一个咽喉要道中断的连锁反应)
    - 集中度风险 (贸易过度集中于少数咽喉要道)
    - 脆弱性指数 (国家层面的综合风险)
    """
    # 集中度风险 (Herfindahl-Hirschman Index)
    df['concentration_risk'] = (
        df['v_canal'] / df['v_total']
    ).pow(2).groupby('iso3').sum()
    
    # 脆弱性指数
    df['vulnerability_index'] = (
        df['trade_at_risk_v'] / df['v_canal'] *
        df['total_economic_cost'] / df['revenue_USD']
    )
    
    return df
```
