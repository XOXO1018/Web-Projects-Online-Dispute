## 📝 技术细节

### 为什么使用 scatter 而不是 geopandas.plot？

1. **更好的控制**: scatter 可以精确控制每个点的大小、颜色、边缘
2. **性能更好**: 对于点数据，scatter 比 GeoDataFrame.plot 更快
3. **兼容性更好**: 不依赖 GeoPandas 的复杂渲染逻辑

### 坐标提取的正确方式

```python
# ❌ 错误：Series 没有 x/y 属性
cp_summary['geometry'].x

# ✅ 正确：使用 apply 提取每个 geometry 的 x/y
cp_summary['geometry'].apply(lambda geom: geom.x)
```

### CRS 坐标系统处理

```python
# 确保世界地图有正确的 CRS
if world.crs is None:
    world = world.set_crs(epsg=4326)

# 咽喉要道数据也使用相同的 CRS
cp_gdf = gpd.GeoDataFrame(..., crs="EPSG:4326")
```

## ⚠️ 关于模拟数据

### 为什么进口依赖度会>100%？

这是因为当前使用**模拟数据**而非真实贸易数据：

1. **贸易流量估算**: 基于 GDP 的固定比例（0.01%）
2. **总贸易估算**: 基于 GDP 的另一个比例（0.02%）
3. **结果**: 某些国家的 `v_canal_import` 可能超过 `v_total_import`

### 如何修复（使用真实数据）

```python
# 从 UN Comtrade 下载真实贸易数据
import pandas as pd

# 真实进口数据
real_imports = pd.read_csv('un_comtrade_imports.csv')

# 真实咽喉要道贸易
chokepoint_trade = pd.read_csv('chokepoint_trade.csv')

# 计算真实依赖度
dependency = chokepoint_trade / real_imports
```
